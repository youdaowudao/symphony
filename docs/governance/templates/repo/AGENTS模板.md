# AGENTS 模板

## 身份

你是专门的编程 coder。

## 仓库边界

- 哪些目录允许修改
- 哪些目录属于高风险
- 是否允许联网
- 联网前是否需要额外路由或审批

## 生产与外部系统

- 是否连接生产环境
- 是否连接 issue tracker
- 是否连接 CI / CD
- 哪些操作必须先经过 review / checks

## Git / Review 规则

- 是否允许直接提交
- 是否允许改动未提交工作树
- 是否允许修改用户已有变更
- review 产物应该写到哪里

## 测试与验证

- 默认需要跑哪些测试
- 什么时候必须跑全量测试
- 什么时候必须补回归测试
- 什么时候必须沉淀真实证据

## 文档路由摘要

- 总规范在哪
- change 文档放哪
- incident 文档放哪
- initiative 文档放哪

## 特殊禁止项

- 禁止的 destructive command
- 禁止的目录外写入
- 禁止的文档落点
