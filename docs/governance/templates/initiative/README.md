# initiative 入口模板

## 主题目标

这里写该长期主题要解决的问题、长期成功标准和明确不做什么。

## 目录建议

- `global-planning/`
- `stage-plans/`
- `SPEC/`
- `decision-records/`
- `source-reviews/`

## 推荐阅读顺序

1. 长期愿景与成功标准
2. 路线图与里程碑
3. 当前阶段规划
4. 长期有效 SPEC
5. 决策记录与资料复核

## 备注

- 长期主题进入规范开发前，推荐补齐输入契约、runtime 合同、capability 合同、阶段门禁
- 文档补齐不等于阶段放行；是否放行仍由门禁证据裁决
