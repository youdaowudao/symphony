# 长期主题入口

这个目录放仓库长期主题、路线图、阶段规划和长期 SPEC。

初始化阶段先保留四个入口：

1. `global-planning/`：长期愿景、成功标准、路线图、里程碑
2. `stage-plans/`：阶段规划、阶段门禁、交接要求
3. `decision-records/`：长期主题里的关键裁决与路线冻结
4. `SPEC/需求阶段分析详细要求.md`：需求分析与 PLAN 阶段的详细要求

推荐阅读顺序：

1. [SPEC/需求阶段分析详细要求.md](./SPEC/需求阶段分析详细要求.md)
2. [global-planning/README.md](./global-planning/README.md)
3. [decision-records/README.md](./decision-records/README.md)
4. [stage-plans/README.md](./stage-plans/README.md)

说明：

1. `global-planning/` 放长期目标与路线图。
2. `stage-plans/` 放阶段规划与阶段级需求。
3. `SPEC/` 放长期有效、可被实现和 review 直接引用的合同。
