# 多项目共享执行池讨论工作区

这里收纳“多项目共享执行池”相关的讨论材料、方案对比、配置草案和可视化说明。

当前使用方式：

1. 这仍然处于讨论期，不代表已冻结为长期规范。
2. 同一主题的材料尽量只放在这里，避免散落到多个长期目录。
3. 讨论收敛后，再把稳定合同拆到 `SPEC/`，把阶段决定保留在这里或上提到 `decision-records/`。
