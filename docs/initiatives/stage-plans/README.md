# 阶段规划

这里放阶段级规划、阶段门禁和交接要求，也可以暂存正在进行中的讨论材料。

使用方式：

1. 讨论还没收敛时，把同一主题的正文、方案对比、配置草案和可视化说明先放在一个子目录里，避免散到多个长期目录。
2. 讨论收敛后，再把稳定合同拆到 `SPEC/`，阶段决定留在这里，长期路线图再视情况回流 `global-planning/`。
3. 当前还在讨论中的主题建议单独开子目录，并在子目录里保留一个入口 README 作为唯一索引。

当前初始化阶段先保留入口，后续再按阶段补充。
