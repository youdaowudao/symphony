- 首页当前 allowlist 是什么。

- 当前是 `go`、`stop`，还是“允许继续无链接 UI，但最终 closeout 被阻断”。

- `stale` 语义直接复用 `RuntimeStatus.classify/2` 的现有 `10` 分钟阈值。
